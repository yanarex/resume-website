# Tony Arrington Portfolio Site

This is a static portfolio website that can be deployed directly with GitHub Pages.

## Files
- `index.html` — main page content
- `styles.css` — styling and layout
- `script.js` — mobile nav + reveal animations
- `assets/` — place your photos and resume file here

## Before publishing
1. Replace these placeholder images in the `assets/` folder or update the image filenames in `index.html`:
   - `profile-placeholder.jpg`
   - `about-placeholder.jpg`
   - `research-placeholder.jpg`
2. Add your actual resume PDF as:
   - `assets/Tony_Arrington_Resume.pdf`
3. Replace the GitHub link in the contact section.

## GitHub Pages
Upload these files to a GitHub repository and enable GitHub Pages for the main branch.
